## Real-World Use Case
File explorer interfaces in IDEs like Visual Studio Code and IntelliJ show a left-side view of the project directory tree -- only the first visible node at each depth level when looking from the left. This is used to generate collapsed folder previews and breadcrumb navigation, showing the leftmost item per level as a quick summary of the tree structure.
