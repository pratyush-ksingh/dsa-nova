## Real-World Use Case
Splunk's log analysis tool uses the minimum-window-subsequence algorithm to find the shortest log segment containing a sequence of events in order. When investigating incidents, SREs search for patterns like 'connection opened -> timeout -> retry -> failure' and the algorithm finds the tightest time window where this event sequence occurred, pinpointing the root cause.
