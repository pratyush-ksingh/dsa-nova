## Real-World Use Case
Database recovery tools at Microsoft SQL Server detect and repair corrupted B-tree indices where two nodes have been accidentally swapped due to a hardware fault. The recover-BST algorithm identifies the two misplaced nodes via an inorder traversal anomaly and swaps them back, restoring index integrity without a full rebuild.
